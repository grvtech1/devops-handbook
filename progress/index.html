    Capstone I Checklist¶

    These are the concrete "done" criteria. Nothing counts until all 6 are ticked.

    • App runs locally — URL Shortener responds to requests, no errors in logs
    • Dockerised — docker build succeeds; container runs and serves traffic
    • Deployed to Kubernetes — Deployment + Service created; kubectl get pods shows Running
    • CI pipeline green — Push to main triggers pipeline; all stages pass ✅
    • ArgoCD synced — App synced from Git repo; Argo UI shows Healthy + Synced
    • Rollback tested — Introduced a bad deploy; ArgoCD / pipeline reverted successfully
    • Cluster cleaned up — Resources deleted; no orphaned namespaces or PVCs left behind
